# 🚀 The Prompting Roadmap

An interactive, pedagogical journey through 12 essential AI prompt engineering techniques. Built for beginners and power users alike to master the art of talking to LLMs.

![Project Preview](https://github.com/keerthana-b-v/prompt-recap/raw/main/assets/preview_placeholder.png) 

## 🌟 Overview

**The Prompting Roadmap** is a "Modern Journal" styled educational platform. It guides users through 12 prompt engineering strategies, ranging from simple instructions to complex reasoning trees (ToT) and secure prompting.

### Key Features
- **Interactive Roadmap**: A visual arc that maps techniques by difficulty (Beginner → Intermediate → Advanced).
- **Comparative Analysis**: Every technique features a side-by-side **Before vs. After** view, showing real AI responses to demonstrate the technique's value.
- **Pedagogical Layer**: Each card includes a "Reality Check" (humorous context), "Daily Use Case", and a "Personal Sneaky Spin".
- **Compare Mode**: Select any two techniques to see how they differ in strategy and output.
- **Modern Journal Aesthetic**: A clean, responsive design using a dot-grid background, elegant typography, and subtle micro-animations.

## 🛠️ Tech Stack
- **Frontend**: Pure HTML5, CSS3 (Vanilla), and JavaScript (ES6+).
- **Design**: Modern Journal theme with Google Fonts (Inter, Architects Daughter).
- **Deployment**: Optimized for GitHub Pages / Vercel.

## 🧠 Techniques Covered
1. **Instruction Prompting** (Beginner)
2. **Role Prompting** (Beginner)
3. **Few-Shot Prompting** (Beginner)
4. **Chain of Thought (CoT)** (Intermediate)
5. **Self-Consistency** (Intermediate)
6. **Prompt Chaining** (Intermediate)
7. **ReAct Prompting** (Advanced)
8. **Function Calling** (Advanced)
9. **Meta Prompting** (Advanced)
10. **Tree of Thoughts (ToT)** (Advanced)
11. **XML Structured Prompts** (Intermediate)
12. **Prompt Injection Security** (Advanced)

## 🚀 Getting Started

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/keerthana-b-v/prompt-recap.git
   ```
2. Open `index.html` in your browser or use a local server:
   ```bash
   npx serve .
   ```

### Deployment
This project is ready for **GitHub Pages**:
1. Push to your main branch.
2. Go to **Settings > Pages**.
3. Select **main** branch and save.
